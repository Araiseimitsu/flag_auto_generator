## プロジェクト概要

本プロジェクトは、工程内検査シート（Excel）に対して、工具ごとの測定No対応に基づき「依頼」判定の数式を自動生成し、出力用xlsxを作成するツールです。

## 目的

- 検査シートの依頼判定を手入力せず自動化する
- 工具と測定Noの対応をJSON設定として再利用可能にする

## 主な機能

- Tkinter UI で設定JSON（シート名、範囲、工具一覧、対応測定Noなど）を作成/編集/保存/読込
- 設定に基づき、指定シートのフラグ列へ `IF(OR(...),"依頼","")` の数式を投入
- 工具名の書き込みと工具行の自動割り当て

## 技術

- Python 3.x
- Tkinter（UI）
- openpyxl（Excel操作）
- JSON（設定管理）
